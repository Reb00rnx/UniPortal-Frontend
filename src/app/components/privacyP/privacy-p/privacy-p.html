Privacy Policy

This policy describes how the University Portal manages and protects your data. As an educational project, we prioritize transparency and security.

1. Data We Collect

We process only the data necessary for the academic system's functionality:

  • Personal Information: First name, last name, index number, and system role (Student or Teacher).
  • Authentication Data: E-mail address and hashed passwords used for secure login.
  • Academic Data: Course enrollment, course codes, and academic grades (values and descriptions).

2. How We Use Your Data

Your information is used strictly to:

  • Authorize access via JWT (JSON Web Tokens).
  • Handle the course enrollment and grade management logic in the database.
  • Generate academic performance reports for students and teachers.
  • Filter and display courses and grades assigned to your specific User ID.

3. Security Measures

To protect your privacy, we implement:

  • Password Hashing: Passwords are never stored in plain text.
  • Stateless Security: We use Spring Security filters to protect every API request.
  • Role-Based Access Control (RBAC): Grading actions are restricted to users with the 'Teacher' role.

4. Contact

If you have any questions regarding this project, feel free to use the Contact tab.
